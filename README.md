# Library Management System API

This is a simple Library Management System API built using Golang and MongoDB. The API allows you to manage books in a library.

## Features

- Retrieve all books
- Retrieve a book by its name

## Technologies Used

- Golang
- MongoDB
- Gorilla Mux
- GoDotEnv

## Getting Started

### Prerequisites

- Go 1.16 or higher
- MongoDB
- Git
